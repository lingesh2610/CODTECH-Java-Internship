## Description
This project demonstrates how to create a REST API client using Java. The application sends an HTTP GET request to a public API and displays the JSON response.

## API Used
- https://api.agify.io/

## Features
- Sends HTTP request to REST API
- Receives JSON response
- Displays data in console

## Technologies Used
- Java
- HTTPURLConnection
- REST API

## How to Run
1. Compile the program:
   javac RestApiClient.java
2. Run the program:
   java RestApiClient

## Sample Output
The program displays age prediction data based on the entered name.
