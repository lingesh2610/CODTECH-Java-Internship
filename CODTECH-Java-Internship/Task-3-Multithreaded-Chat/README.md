## Description
This project implements a multithreaded chat application using Java sockets. A server handles multiple clients simultaneously using multithreading.

## Features
- Client-server architecture
- Multiple clients can chat at the same time
- Real-time message broadcasting

## Technologies Used
- Java
- Sockets
- Multithreading

## How to Run
1. Compile Server and Client:
   javac Server.java
   javac Client.java

2. Start Server:
   java Server

3. Start multiple Clients:
   java Client
