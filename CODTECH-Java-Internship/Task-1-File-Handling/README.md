## Description
This project demonstrates basic file handling operations in Java. The program allows users to write data to a text file, read data from the file, and modify the file by appending new content.

- Read content from an existing file
- Modify file content by appending text

## Files Included
- FileHandlingUtility.java

## How to Run
1. Compile the Java program:
   javac FileHandlingUtility.java
2. Run the program:
   java FileHandlingUtility

## Output
The program creates and updates a text file named `sample.txt` based on user input.
